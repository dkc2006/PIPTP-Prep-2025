### ✅ **Question (Problem Statement)**

What will be the output of the following pseudocode?

### 🧠 **Pseudo Code**


```
1.  Integer a, b
2.  Set a = 3, b = 3
3.  a = b
4.  if (1 ^ 1)
5.      a = 1
6.  Else
7.      b = 2
8.  End if
9.  Print a + b

```
### 🔍 Step-by-Step Execution:
Step 2: `a = 3`, `b = 3`

Step 3: `a = b` → `a = 3`

Step 4: `1 ^ 1` → Bitwise XOR of 1 and 1 is `0` → **False**

Step 7 (Else executes): `b = 2`

Now:

- `a = 3`

- `b = 2`

### 🖨️ Final Output on Console

`a + b` = `3 + 2` = `5`

`Correct output is:` **5** ✅
